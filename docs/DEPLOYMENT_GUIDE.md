# CipherZB160-IQ Deployment Guide

This document outlines instructions for building and running the frontend dashboard locally or deploying to Vercel and other cloud providers.

## Local Running
1. Serve the `website/` folder using a local web server (e.g. `npx serve website` or VS Code Live Server).
2. Open `http://localhost:3000` or the corresponding local server URL in your browser.

## Vercel Deployment
The repository includes a [vercel.json](file:///c:/Users/akash/Downloads/Mule%20Account%20Detection%20and%20Classification/deployment/vercel.json) configuration under the `deployment/` directory.

To deploy on Vercel:
1. Connect this repository to your Vercel Account.
2. Configure the build root settings to point to the `website/` directory.
3. Deploy.
